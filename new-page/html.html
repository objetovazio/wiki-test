Bem-vindo ao Sistema LiberFly

Introdução

O sistema LiberFly foi desenvolvido para ajudar pessoas com problemas relacionados a viagens. Através do nosso processo, conseguimos capturar dados de leads, realizar análises preliminares e dar seguimento ao atendimento personalizado para cada caso.

Principais Componentes

  • LiberFly-Website-BR: Interface inicial em Nuxt3, onde o lead se registra e compartilha informações.
  • LiberFly-Backend: API em Laravel responsável por capturar dados dos leads e distribui-los para os atendentes.
  • LiberFly-Sales-Document-API: Gerencia a documentação dos leads, integrando com CRM e ClickSign para assinatura digital.
  • Banco de Dados: Repositório central para armazenar todas as informações dos leads.

Workflow do Sistema

O fluxo de trabalho começa com o registro do lead, passando por uma análise preliminar realizada no frontend. Após isso, o backend distribui os dados para o atendimento, onde a Sales-Document-API gerencia a documentação e as assinaturas digitais.

Integrações Externas

  • CRM: Centraliza as informações do lead e o andamento dos casos.
  • ClickSign: Plataforma de assinatura digital para formalização de contratos.
